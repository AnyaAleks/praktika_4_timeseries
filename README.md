# TimeSeries Tourism Analyzer: Austria & Belgium (2000-2024)

## Проект по анализу временных рядов туристических потоков

### Описание проекта
Программный комплекс для комплексного анализа временных рядов, характеризующих количество отправных туристических поездок из Австрии и Бельгии за период 2000-2024 гг. Проект реализован в рамках учебной практической работы по курсу "Статистическая обработка информации".

### Основные возможности анализа
- **Предварительная обработка данных**: загрузка, очистка и визуализация временных рядов
- **Статистический анализ**: проверка на стационарность (ADF, KPSS тесты), анализ структурных разрывов
- **Корреляционный анализ**: оценка связи между рядами разных стран
- **Моделирование трендов**: подбор оптимальных моделей (линейные, полиномиальные, ARIMA)
- **Прогнозирование**: построение прогнозов на 5 лет вперед с доверительными интервалами
- **Сравнительный анализ**: параллельное исследование динамики двух стран

### Ключевые результаты исследования
1. **Нестационарность рядов**: подтверждена тестами Дики-Фуллера и KPSS
2. **Структурные разрывы**: статистически значимые изменения в 2020 году (влияние COVID-19)
3. **Сильная корреляция**: r = 0.872 между туристическими потоками Австрии и Бельгии
4. **Оптимальные модели**: ARIMA(0,1,1) для обеих стран
5. **Дифференцированное восстановление**: Австрия (+38% к 2019) vs Бельгия (+3.4% к 2019)

### Технологический стек
- **Python 3.12+** с основными библиотеками:
  - `pandas`, `numpy` - обработка данных
  - `statsmodels`, `scipy` - статистический анализ
  - `matplotlib`, `seaborn` - визуализация
  - `scikit-learn` - метрики качества моделей

2. Установите зависимости:

###  Методология исследования
1. Проект реализует полный цикл анализа временных рядов:
2. Сбор данных из официальных источников (UN Tourism)
3. Разведочный анализ и визуализация
4. Проверка статистических гипотез
5. Моделирование и прогнозирование
6. Интерпретация результатов и формирование выводов

### Теоретическая база
1. Международные рекомендации по статистике туризма 2008 (IRTS 2008)
2. Методы анализа временных рядов (Бокс-Дженкинс)
3. Статистические тесты: Дики-Фуллера, KPSS, тесты на структурные изменения
4. Методы машинного обучения для прогнозирования
